## What is BRAID?

BRAID stands for **B**udget, **R**isks, **A**ctions, **I**ssues, **D**ecisions - the core tracking elements for project management.

braidMgr provides:

- **Multi-tenant architecture** - Separate databases per organization
- **RAID log management** - Track risks, actions, issues, and decisions
- **Budget tracking** - Monitor project budgets and actuals
- **AI-powered chat** - Query project data using natural language
- **Timeline views** - Visualize project schedules and milestones
